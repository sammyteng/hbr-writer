# 📚 哈佛商业评论 Writer Skill

> 让 AI 以哈佛商业评论（HBR）「管理框架+案例驱动」风格写管理洞察文章的可复用 Skill。

## 简介

本 Skill 基于对哈佛商业评论中文版管理类文章的结构化解析，将 HBR「理论框架+企业案例+可操作建议」的标准写法标准化为 AI 写作指令。

## 文件结构

```
hbr-writer/
├── SKILL.md                          # 主技能文件
└── resources/
    ├── playbook_leadership.md        # 骨架一：领导力洞察
    ├── playbook_strategy.md          # 骨架二：战略决策框架
    ├── playbook_innovation.md        # 骨架三：创新与变革管理
    ├── playbook_talent.md            # 骨架四：人才与组织
    └── playbook_digital.md           # 骨架五：数字化转型管理
```

## HBR 核心写作风格

| 维度 | 特征 |
|------|------|
| 核心价值 | 每篇必须有可命名的「管理框架」 |
| 论据结构 | 理论主张 → 企业案例 → 可操作建议 |
| 读者层级 | 企业高管、中层管理者、商学院学生 |
| 语气 | 权威专业，但非学术晦涩 |
| 禁忌 | 没有框架只有观点、建议太抽象 |

## 核心写作哲学

- **框架化思维**：把管理问题「模型化」，让读者带走一个可用的分析工具
- **案例驱动**：每个论点都要有来自真实企业的具体验证
- **实践导向**：理论要落地，必须给出「你应该怎么做」的具体步骤
- **本地化**：中文版要结合中国商业实践，不只是翻译西方内容

## 如何使用

```
使用 hbr-writer，帮我用哈佛商业评论风格写一篇关于 [管理议题] 的洞察文章
```

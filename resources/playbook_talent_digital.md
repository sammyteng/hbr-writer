# 骨架四：人才与组织

适用场景：为管理者提供人才选拔、团队建设、组织设计的框架洞察。

## 结构模板

```
【打破错误的人才直觉】
「[常见的人才判断标准]——听起来很有道理。但这正是大多数管理者在人才问题上最常犯的错。」

【人才评估框架】
建立一个多维度人才评估模型：
- 通常包含：能力（Competence）× 潜力（Potential）× 价值观（Values）
- 给每个维度清晰定义（避免主观解读）
- 提供评估工具/问题清单

【高潜人才的识别方法】
「研究发现，高潜人才有[N]个独特特征：」
- [特征1]：如何识别 + 企业案例
- [特征2]：如何识别 + 企业案例
- [特征3]：如何识别 + 企业案例

【人才管理中的中国特色挑战】
针对中文读者的本土化内容：
- 中国企业文化下的人才困境（关系vs.能力）
- 90/00后职场期待与管理者代际差异
- 本土企业如何吸引海归/跨国公司人才

【建立人才梯队的3步行动】
具体可操作的建议，每步给出实施标准
```

# 骨架五：数字化转型管理

适用场景：帮助企业领导者理解数字化转型的本质挑战和推进路径。

## 结构模板

```
【数字化转型的本质误解】
「大多数企业把数字化转型理解为[错误理解]。这是为什么大多数转型项目停留在表面的原因。」

【数字化转型的「本质层次」框架】
建立转型成熟度模型：
- 层次1：数字化效率（流程自动化）
- 层次2：数字化体验（用户侧）
- 层次3：数字化商业模式（本质重构）
大多数企业在哪个层次上卡住了？

【成功案例深度拆解】
选1家在中国商业语境里有代表性的转型案例：
- [公司]在转型前的困境是什么？
- 他们如何定义「转型成功」？
- 经过[N年]，实现了什么具体改变？

【CEO视角 vs. IT团队视角 vs. 员工视角】
三个视角的差异和对齐方式：
数字化转型失败的核心原因往往是「愿景没有对齐」

【推进路径的建议】
分阶段、可操作的行动路线图：
Phase 1（0-6个月）：[目标] + [关键行动]
Phase 2（6-18个月）：[目标] + [关键行动]
Phase 3（18个月+）：[目标] + [关键行动]
```
